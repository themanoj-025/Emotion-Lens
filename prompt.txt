## 🧭 PROJECT CONTEXT

You are building a **10/10 production-grade Streamlit dashboard** for a Face Emotion Detection system powered by a CNN trained on FER2013. The model (`emotion_model.h5`) detects 7 emotions: **Angry, Disgust, Fear, Happy, Neutral, Sad, Surprise** from 48×48 grayscale face images.

The existing codebase has:
- `train.py` — CNN training with CLI args (epochs, batch_size, model_name)
- `inference.py` — Static image prediction
- `webcam_inference.py` — Real-time OpenCV webcam detection
- `emotion_model.h5` — Pre-trained model (~62% val accuracy on FER2013)
- `requirements.txt` — tensorflow, numpy, matplotlib, kagglehub, opencv-python, pillow

**streamlit-webrtc Implementation Template:**
```python
from streamlit_webrtc import webrtc_streamer, VideoProcessorBase
import av

class EmotionProcessor(VideoProcessorBase):
    def __init__(self):
        self.model = load_model_cached()
        self.face_cascade = cv2.CascadeClassifier(...)
        self.result = None  # Store last prediction for display
    
    def recv(self, frame: av.VideoFrame) -> av.VideoFrame:
        img = frame.to_ndarray(format="bgr24")
        gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
        faces = self.face_cascade.detectMultiScale(gray, 1.3, 5)
        
        for (x, y, w, h) in faces:
            roi = gray[y:y+h, x:x+w]
            roi = cv2.resize(roi, (48, 48))
            roi = roi.astype('float32') / 255.0
            roi = np.expand_dims(roi, axis=[0, -1])
            preds = self.model.predict(roi, verbose=0)[0]
            emotion = EMOTIONS[np.argmax(preds)]
            conf = preds.max()
            
            # Draw colored bounding box
            color = hex_to_bgr(EMOTION_CONFIG[emotion]['color'])
            cv2.rectangle(img, (x, y), (x+w, y+h), color, 2)
            
            # Draw label with background
            label = f"{EMOTION_CONFIG[emotion]['emoji']} {emotion} {conf*100:.0f}%"
            (tw, th), _ = cv2.getTextSize(label, cv2.FONT_HERSHEY_SIMPLEX, 0.7, 2)
            cv2.rectangle(img, (x, y-th-15), (x+tw+10, y), color, -1)
            cv2.putText(img, label, (x+5, y-5), cv2.FONT_HERSHEY_SIMPLEX, 0.7, (255,255,255), 2)
            
            self.result = {'emotion': emotion, 'confidence': conf, 'all_preds': preds}
        
        return av.VideoFrame.from_ndarray(img, format="bgr24")

ctx = webrtc_streamer(
    key="emotion-detection",
    video_processor_factory=EmotionProcessor,
    rtc_configuration={"iceServers": [{"urls": ["stun:stun.l.google.com:19302"]}]},
    media_stream_constraints={"video": True, "audio": False},
)

if ctx.video_processor and ctx.video_processor.result:
    result = ctx.video_processor.result
    # Render live emotion metrics below the video
    render_emotion_dashboard(result)
```

## 🔧 UTILITY MODULES

### `utils/model_utils.py`
```python
@st.cache_resource
def load_model_cached(path='emotion_model.h5'):
    """Load model once, cache in session for speed."""
    return load_model(path)

@st.cache_resource  
def load_face_cascade():
    return cv2.CascadeClassifier(
        cv2.data.haarcascades + 'haarcascade_frontalface_default.xml'
    )
```

### `utils/session_utils.py`
```python
def init_session_state():
    defaults = {
        'predictions': [],          # List of {emotion, confidence, timestamp}
        'snapshots': [],            # Captured frames as PIL Images
        'session_start': datetime.now(),
        'total_predictions': 0,
        'game_score': 0,
    }
    for k, v in defaults.items():
        if k not in st.session_state:
            st.session_state[k] = v

def add_prediction(emotion, confidence, all_probs):
    st.session_state.predictions.append({
        'emotion': emotion,
        'confidence': confidence,
        'probabilities': all_probs.tolist(),
        'timestamp': datetime.now().isoformat()
    })
    st.session_state.total_predictions += 1
```

Allow users to upload their own labeled face images, auto-generate a mini training session, and compare with base model accuracy.

### Feature 9: Grad-CAM Live
In real-time mode, optionally overlay Grad-CAM heatmap on the face region to visualize which facial features (eyes, mouth, brow) the CNN focuses on per emotion.
